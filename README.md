# HP2803A

A two-channel instrument that uses quartz crystal resonators with a linear frequency-temperature characteristic to measure temperature to a maximum resolution of 0.0001 Â°C.

## Architecture

The firmware is written in C and targets an AVR ATmega 644A microcontroller with an A64-based simulator for development and testing.
