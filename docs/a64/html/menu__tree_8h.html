Detailed Description

Declarations of the menu text and menu node arrays for the navigation system.

This header provides access to the global menu data structures used by the menu navigation system. It defines:

  • menu_text[]: an array of text labels for menu items.
  • menu[]: an array of MenuNode structures that describe the tree of menu items.
Note
The enumeration values for node IDs, text IDs, and action IDs are declared in menu_types.h and are not repeated here.
