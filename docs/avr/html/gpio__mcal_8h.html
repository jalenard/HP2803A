◆ gpio_set_timebase()

void gpio_set_timebase ( TimebaseOption  timebase)

Configure the timebase selection lines PB1 (TB-A) and PB2 (TB-B).

Updates PB1 and PB2 on PORTB according to the requested timebase mode. If the given timebase value is not recognized, the function safely does nothing.

Parameters
timebaseOne of:
  • TIMEBASE_MSEC100 → 10 Hz (0.1 s)
  • TIMEBASE_SEC1 → 1 Hz (1 s)
  • TIMEBASE_SEC10 → 0.1 Hz (10 s)
