◆ timer_mcal_init()

void timer_mcal_init ( void  )

Configures Timer 0 and Timer 1 in CTC mode, sets their prescalers and compare values, and enables interrupts for the required timing operations.

Note
The complete low-level (re-)initialization sequence for Timer 0 is:
uint8_t s = SREG;
cli();
TIMSK0 &= ~(1 << OCIE0A);
TCCR0B = 0;
TCCR0A = (1 << WGM01);
OCR0A = 249;
TCNT0 = 0;
TIFR0 = (1 << OCF0A);
TIMSK0 |= (1 << OCIE0A);
TCCR0B = (1 << CS02);
SREG = s;
