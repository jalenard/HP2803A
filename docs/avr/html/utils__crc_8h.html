Compute a 16-bit CRC (CRC-16/CCITT-FALSE variant).

Implements the standard CRC-16-CCITT algorithm using polynomial 0x1021 (x^16 + x^12 + x^5 + 1), initial value 0xFFFF, and MSB-first bit processing. Each input byte is XORed into the upper byte of the 16-bit register and shifted bit-by-bit through the feedback taps.

Parameters
dataPointer to the input data buffer.
lenNumber of bytes to process.
Returns
16-bit CRC remainder.
