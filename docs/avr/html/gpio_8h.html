Detailed Description

Provides safe masked writes to AVR ports using shadow registers.

This module allows writing to selected bits of PORTA..PORTD without affecting other bits, using internal shadow variables.

Update selected bits of PORTA without affecting other bits.

Example usage:

#include "gpio.h"
uint8_t counter_num = 5;
// Set PA7 (enable) and update PA2:0 (counter select) without touching PA3:6
gpio_a_write_masked((1 << PA7) | 0x07, // i.e. 0b10000111
(1 << PA7) | (counter_num & 0x07));
// Retrieve the current shadow value of PORTA
uint8_t current_shadow = gpio_a_get_shadow();
